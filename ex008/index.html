Formatações de texto

Negrito

Para colocar em negrito so usar a tag b (não semantico)

Para colocar palavras em negrito é so usar a tag STRONG (semantico)

Italico

Para colocar em Italico so usar a tag i (não semantico)

Para colocar palavras em Italico é so usar a tag em (semantico)

Texto Marcado

Para marcar uma parte do texto utilizamos a tag MARK, por exemplo com esse termo

Texto Grande

Para colocar o texto grande, tem a tag big, porem ela nao ta em uso mais.

Texto Pequeno

Para colocar o texto pequeno, tem a tag small ela ta em uso ainda.

Texto Sobrescrito

Para colocar um texto Sobrescrito, utilizamos a tag sup, como exemplo x20+10

Texto Subscrito

Para colocar um texto Subscrito, utilizamos a tag sub, como exemplo H2O
